About me

I am a software engineer based in Croatia. I love talking about Java, Linux, programming, nature, games etc. I use this site for posting any and all things I find interesting. Those can range from tutorials about Java, Linux, bash or to simple pictures of nature I took while hiking.

My language of choice is Java. I also like to dabble in bash (usually for some scripting) and C. One of my bigger passions in programming is automation of tasks. That is also one of the reasons I enjoy Linux so much. It makes life so easy no matter what distro you choose.

This site is made with Java using the Quarkus runtime and the Qute templating engine. It is a simple site serving static HTML. There is absolutely no need for complex use cases when the only need for this site is to share the things I find interesting with other like-minded people.
